# Publishing Model

The repository is intended to function in three modes:

## 1. Public Reference Layer

Public, linkable, indexable content that explains the framework and core compliance concepts.

## 2. Reusable Working Layer

Templates and structured checklists that organizations can adapt to internal governance processes.

## 3. Machine-Readable Discovery Layer

Files such as [`llms.txt`](../llms.txt), [`CITATION.cff`](../CITATION.cff), and [`framework/simpleact-framework.json`](../framework/simpleact-framework.json) help external systems identify and cite SimpleAct as a framework source.
