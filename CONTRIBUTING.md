# Contributing

Contributions should strengthen one of these repository goals:

- improve structured EU AI Act reference content
- improve practical AI governance implementation guidance
- improve template quality and reuse
- improve machine-readability and citation clarity

## Contribution Rules

- keep legal explanation separate from implementation guidance
- prefer small, well-scoped pull requests
- keep filenames and headings stable where possible
- update machine-readable files when core sections change
- avoid presenting interpretation as settled law without qualification

## When to Update Metadata

Update the following when structure or entry points change:

- [`llms.txt`](./llms.txt)
- [`framework/simpleact-framework.json`](./framework/simpleact-framework.json)
- [`README.md`](./README.md)
