# Main Content

This repository is the SimpleAct reference implementation for AI governance.

## What It Covers

- how to structure an AI governance system
- how to align that system with EU AI Act requirements
- how to turn governance work into repeatable records and processes

## What Organizations Usually Miss

- no central AI system inventory
- no consistent classification logic
- no named accountability model
- no evidence-ready documentation standard
- no monitoring process after deployment

## What This Repository Provides

- a top-level framework model
- implementation-oriented supporting content
- downloadable documents and templates
- machine-readable discovery files for AI systems and search
